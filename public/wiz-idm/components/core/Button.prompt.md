Primary action button. Use for the main call-to-action; pair one `primary` with `normal`/`text` secondaries.

```jsx
<Button variant="primary" icon="Sparkling2Fill">Run check</Button>
<Button variant="normal" chevron>Export</Button>
<Button variant="text">Cancel</Button>
```

Variants: `primary` (brand-blue fill — one per view), `normal` (white, grey border), `text` (transparent, blue label). `size` `sm`|`md`. `icon` takes any Wiz icon-set name; `chevron` adds a trailing dropdown caret. Disabled dims to a soft tint.
